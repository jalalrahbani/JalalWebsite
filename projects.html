Featured Research Tools

I develop custom Python applications for biological data analysis and image processing. These tools are designed to address specific challenges in cell biology research and are available for free use by the research community. Each tool is available in two formats: a web-based version for quick access and the original desktop application for full functionality.

3D Cell Analysis Tool

Description: A sophisticated 3D cell analysis and visualization tool for processing biological volumes, performing machine learning segmentation, and creating interactive 3D visualizations. This tool provides advanced capabilities for analyzing confocal microscopy data and 3D cell structures. Features include multiple segmentation methods, interactive ROI selection, brightness/contrast controls, surface management, and comprehensive volume analysis. Available as both a web application and a native PyQt5 desktop application with GPU acceleration.

Features:

  • 3D image processing and multi-dimensional support
  • Interactive ROI selection with freeform drawing
  • Machine learning segmentation using Random Forest
  • 3D surface rendering and volume visualization
  • Background correction and image enhancement
  • Volume analysis and quantitative metrics
  • Export capabilities for processed data
  • Support for TIFF, TIF, PNG, JPG, JPEG formats

Technology Stack: Python, Streamlit, VTK, OpenCV, NumPy, scikit-learn, tifffile, Plotly

Cell Migration Analysis Tool

Description: An advanced cell migration analysis tool for processing and analyzing cell tracking data from time-lapse microscopy experiments. This tool provides comprehensive analysis of cellular movement patterns, trajectory analysis, and statistical comparisons. Features include Excel file processing, comprehensive statistical analysis (t-tests, Mann-Whitney U), multiple visualization types, advanced metrics (MSD, directionality over time/distance), and detailed export capabilities. Available as both a web application and a native Tkinter desktop application with real-time analysis.

Features:

  • Automated cell tracking and trajectory analysis
  • Velocity and persistence calculations
  • Statistical analysis of migration patterns
  • Interactive visualization of cell trajectories
  • Export capabilities for publication-ready figures
  • Multi-track analysis capabilities
  • Time-lapse microscopy data processing

Technology Stack: Python, OpenCV, NumPy, Pandas, Matplotlib, scikit-image, Streamlit
